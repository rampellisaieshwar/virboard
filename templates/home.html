Features

Hand Tracking

The project's core functionality is hand tracking, allowing users to draw on a virtual whiteboard by moving their hands in front of the camera. This feature enables intuitive and interactive drawing.

Real-Time Drawing

The application provides real-time drawing capabilities, making it suitable for creative and educational purposes. Users can see their drawings as they create them.

FPS Display

Displaying the frames per second (FPS) on the screen is a valuable feature as it provides feedback on the application's performance and responsiveness.

Saving Drawings

Users can save their drawings by pressing the 's' key. This feature is essential for preserving artwork or notes created on the virtual whiteboard.

Clearing the Whiteboard

The ability to clear the virtual whiteboard by pressing the 'x' key is a useful feature for starting a new drawing without restarting the application.

Database Integration

The project integrates with a MySQL database to store information about saved images. While it may not be directly visible to the user, it's a valuable feature for organizing and retrieving saved drawings.
